== Description ==
A widget that shows the latest post that contains video embedded with iframe.


Features:

- Shows the latest video embedded with iframe.
- Link to post title containing the video.
- Link to category containing the video post.
- Responsive, compatible with small devices.
- Very lightweight.


== Installation ==
1. Upload `latest-video-post-widget` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin.
3. Activate the Latest Video Post Widget from the 'Plugins' menu.
4. Add the widget to your sidebar from the 'Widgets' design page.
5. Configure the widget according to your needs.
